## 🧩 What this is

the-vibe-pm is a set of skills, templates, and guides for people building with AI agents. It is made for product people, not engineers.

Use it to plan work, write better prompts, shape agent tasks, and keep AI output useful. It helps you move from idea to a working draft with less guesswork and less back and forth.

## 📦 What you get

This project is built around practical files you can use right away:

- Clear guides for working with AI agents
- Templates for product planning
- Prompt patterns for common tasks
- Checklists for shaping work before you hand it to an AI tool
- Simple steps for turning rough ideas into useful output

It is set up to help you think through the work first, then use AI with more control.
